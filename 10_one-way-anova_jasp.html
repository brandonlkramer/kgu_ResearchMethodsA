10A.3Assumption Checks

Open the Assumption Checks section in the dialog and check ☑ Homogeneity tests and ☑ Q-Q plot of residuals.

Fig 10–5Levene's test for homogeneity of variances
screenshots/fig10-5.png
Assumption Checks: Levene's test table (F, df1, df2, p)
①

Levene's test result: F(2, 39) = 0.29, p = .750.

Because p > .05, we do not reject the null hypothesis of equal variances. Variances are assumed equal — the parametric ANOVA is appropriate, and Tukey's post-hoc test can be used.

If Levene's test were significant (p < .05), you would switch to the Games–Howell post-hoc test.

Fig 10–6Q-Q plot of residuals
screenshots/fig10-6.png
Q-Q plot: data points falling roughly along the diagonal line
→

A Q-Q plot shows the distribution of your residuals (the differences between each observed score and its group mean) against what you would expect from a normal distribution. Points falling close to the diagonal line indicate approximately normal residuals — meaning the normality assumption is met.

💡 Which Post-Hoc Test to Choose?

Quick Reference

Tukey's HSDEqual variances & equal group sizes (the default — use for Ellis & Yuan data)
Games–HowellLevene's test is significant (unequal variances)
SchefféUnequal group sizes (unbalanced design)
BCa Bootstrap CIsNormality violated — use instead of or alongside p-values

10A.4Output: Omnibus F-Test

The first table in the ANOVA output is the Between Subjects Effects table. This contains the omnibus F-test result.

Fig 10–7Between Subjects Effects — omnibus ANOVA table
screenshots/fig10-7.png
Between Subjects Effects table: Cases, SS, df, MS, F, p, ω²
①

Look at the Group row (your IV). Key values to extract:

• F = 9.05  ·  df = 2, 39  ·  p < .001  ·  ω² = 0.28

Because p < .05, the omnibus test is statistically significant — there is at least one significant difference among the three groups. The omega-squared value of 0.28 indicates that planning condition accounts for approximately 28% of the variance in syntactic variety — a large effect.

The Residuals row shows the within-group variance (SSE = 519.00, df = 39). This is the denominator of the F-ratio.

⚠ The Omnibus Test Is Just the Beginning

A significant omnibus F-test only tells you that some groups differ — not which ones. You must run post-hoc tests to identify where the differences lie. Do not stop here.

10A.5Setting Up Post-Hoc Tests

  1. 1

    In the ANOVA dialog, open the Post Hoc Tests section by clicking the triangle to expand it.

  2. 2

    Move Group from the left panel into the right panel to request post-hoc tests for this variable.

  3. 3

    Under Correction, check ☑ Tukey (variances are equal, so Tukey is appropriate here).

  4. 4

    Check ☑ Effect size to include Hedges' gₛ in the output.

  5. 5

    Check ☑ Confidence intervals to include 95% CIs for each pairwise comparison.

  6. 6

    Optional (recommended): Click From … Bootstraps and set the number of samples to 10,000. Change the CI type to BCa. This produces robust CIs that do not rely on normality assumptions.

Fig 10–8Post Hoc Tests dialog — Group moved across, Tukey & Effect size checked
screenshots/fig10-8.png
Post Hoc Tests panel: Group moved to right, Tukey + Effect size + CI + Bootstrap checked
①

Right panel: Group should appear in the box on the right — this tells JASP to run pairwise comparisons for all levels of Group.

Correction: Tukey is checked. If your variances had been unequal, you would check Games-Howell instead.

Effect size and Confidence intervals are both checked to get Hedges' gₛ and 95% CIs for each comparison.

10A.6Output: Post-Hoc Comparisons

The post-hoc output shows a row for every pair of groups. The most important columns are the mean difference and its 95% CI — if the CI does not contain zero, the difference is statistically significant.

Fig 10–9Post Hoc Comparisons table — all pairwise results
screenshots/fig10-9.png
Post Hoc table: Mean Diff, SE, t, p (Tukey), 95% CI, Hedges' gₛ for NP/PTP/OLP pairs
①

There are three rows — one per pair. Read each 95% CI:

• NP vs. PTP: CI = [3.17, 8.78] — does not contain zero → significant. PTP scored higher.

• NP vs. OLP: CI = [−0.10, 5.34] — contains zero → not significant.

• PTP vs. OLP: CI = [0.93, 5.85] — does not contain zero → significant. PTP scored higher.

Fig 10–10Post Hoc Effect Sizes — Hedges' gₛ with 95% CIs
screenshots/fig10-10.png
Effect Sizes table: Hedges' gₛ point estimate and 95% CI for each pair
①

Hedges' gₛ corrects for small sample sizes and uses the pooled SD as the standardizer. Values for the Ellis & Yuan data:

• NP vs. PTP: gₛ = 1.48 [0.61, 2.24] — large effect

• NP vs. OLP: gₛ = 0.70 [0.11, 1.47] — medium, but CI includes small-to-large range

• PTP vs. OLP: gₛ = 0.92 [0.20, 1.61] — medium-to-large effect

By Plonsky & Oswald's (2014) SLA benchmarks: 0.60 = small, 1.00 = medium, 1.50 = large.

📌 Interpreting the Full Pattern

The PTP group is significantly different from both other groups. The NP and OLP groups are not significantly different from each other. This means the key finding is that pre-task planning produced more syntactic variety than either no planning or online planning, while unlimited writing time (OLP) did not offer a significant advantage over no planning at all.

10A.7Model Write-Up

Model Reporting Paragraph — One-Way ANOVA

To examine whether planning condition affected the syntactic variety of learners' writing, a one-way between-subjects ANOVA was conducted. Descriptive statistics were: no planning (NP), M = 16.21, SD = 4.10, n = 14; pre-task planning (PTP), M = 22.07, SD = 3.58, n = 14; online planning (OLP), M = 18.86, SD = 3.21, n = 14. Levene's test indicated that the equal-variances assumption was met, F(2, 39) = 0.29, p = .750, and the Q-Q plot of residuals did not show serious departures from normality.

The omnibus F-test was statistically significant, F(2, 39) = 9.05, p < .001, ω² = 0.28, indicating that planning condition explained approximately 28% of the variance in syntactic variety. Tukey post-hoc tests revealed that the PTP group produced significantly more syntactic variety than the NP group (mean difference = 5.86, 95% CI [3.17, 8.78], Hedges' gₛ = 1.48 [0.61, 2.24], a large effect) and significantly more than the OLP group (mean difference = 3.21, 95% CI [0.93, 5.85], Hedges' gₛ = 0.92 [0.20, 1.61], a medium-to-large effect). The difference between the NP and OLP groups was not statistically significant (mean difference = 2.64, 95% CI [−0.10, 5.34], Hedges' gₛ = 0.70 [0.11, 1.47]). These findings suggest that pre-task planning time led to meaningfully greater syntactic complexity in writing, while unlimited online planning time did not confer a significant advantage over no planning.
